Capacitación para el jugador

Mantenimiento de jugador

En el mantenimiento de jugador se tiene como objeto registrar un nuevo jugador, sucesivo a eso realizar las siguientes opciones:

Modificar

Eliminar

Buscar

Ayuda

Reporte

Para ingresar un nuevo registro debemos dirigirnos al menú de catalogos en el modulo de catalogos y seleccionar la opción de "Mantenimientos", seguidamente la opción de "Mantenimiento jugador

Despues podremos observar el formulario a llenar para agregar una nueva aplicación.
